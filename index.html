ปัญหาที่ต้องการแก้ไข

Project ต้องการสร้างตัวจับเวลาสำหรับ เกมโชว์ ความบันเทิง การออกกำลังกาย และ การฝึกฝน ความสำคัญของโปรเจคคือ เพื่อจับเวลาให้ตรงความจริงให้ได้มากที่สุด คนที่ได้ประโยชน์ นักออกกำลังกาย ผู้จัดการแข่งขัน นักกีฬา ปัญหานี้แก้ไขได้โดย Project นี้จะใช้ censor ในการตรวจจับและหยุดเวลาทำให้ได้เวลาที่ใกล้เคียงความจริงที่สุด Input เวลาเริ่มต้น ภาพจากกล้อง Output เวลาสิ้นสุด และ ข้อมูลเวลา

ความสำคัญของปัญหา

ปัญหาในการจับเวลาให้แม่นยำมีความสำคัญอย่างยิ่งต่อกิจกรรมหลายประเภทเช่น การแข่งขันต่างๆ การใช้เซ็นเซอร์เป็นแนวทางแก้ไขที่มีประสิทธิภาพสูง เนื่องจากเซ็นเซอร์สามารถให้ข้อมูลที่แม่นยำและรวดเร็ว ทำให้ผลลัพธ์ที่ได้มีความน่าเชื่อถือและสามารถนำไปใช้ประโยชน์ได้อย่างหลากหลาย

ความคืบหน้าโครงการ

รายละเอียดขั้นตอนที่ดำเนินการ

การระบุปัญหา :
เราเริ่มต้นด้วยการค้นคว้าหาข้อมูลว่าปกติแล้วปัจจัยใดที่จะทำให้การฝึกฝนร่างกายนั้นมีประสิทธิภาพที่ดีขึ้น จากการศึกษาพบว่าปัจจัยที่สำคัญได้แก่ การจับเวลาในการออกกำลังกายอย่างแม่นยำ การวัดระดับความเข้มข้นของการฝึก ความสามารถในการติดตามและวิเคราะห์ผลการฝึกฝน รวมถึงการสร้างสภาพแวดล้อมที่ใกล้เคียงเหตุการจริงมากที่สุด


การวางแผนการแก้ปัญหา :
หลังจากเข้าใจปัญหาแล้ว เราวางแผนพัฒนาระบบที่ใช้บอร์ด Arduino และเซนเซอร์วัดระยะทาง เป้าหมายของเราคือสร้างอุปกรณ์ที่สามารถจับเวลาและทำให้เวลาที่วัดได้ใกล้เคียงกับความจริงมากที่สุด


การสร้างต้นแบบ :
ขั้นตอนถัดไปคือการประกอบอุปกรณ์ที่จำเป็น ได้แก่ บอร์ด Arduino เซนเซอร์ ปุ่มกดแบบรายการเกมโชว์ 4 Digit 7 Segment Display และอุปกรณ์อิเล็กทรอนิกส์อื่น ๆ เราทำการเชื่อมต่อและทดสอบวงจรเบื้องต้นเพื่อให้แน่ใจว่าอุปกรณ์ทำงานได้ตามต้องการ


การเขียนโค้ดและการทดสอบ :
เมื่อฮาร์ดแวร์พร้อมแล้ว เราเขียนโค้ดให้ Arduino สามารถทำระบบจับเวลาแบบไปข้างหน้าและนับถอยหลัง และหยุดการทำงานด้วยปุ่มกด หรือเซนเซอร์วัดระยะทาง แล้วนำข้อมูลที่ได้ส่งไปยัง MQTT ซึ่งทดสอบโดยลองจับเวลา และวิ่งผ่านเซนเซอร์เพื่อวัดความแม่นยำของตัวจับเวลา


การแก้ไขปัญหา :
เราพบปัญหาว่าอุปกรณ์ ESP32-CAM ที่เป็นอุปกรณ์หลักในการตรวจจับไม่สามารถใช้งานได้ ในการแก้ไขปัญหานี้ เราได้ปรับไปใช้งาน Distance Ultrasonic Sensor ซึ่ง เซนเซอร์วัดระยะทาง ในการหยุดเวลา


สถานะปัจจุบัน :
ระบบสามารถจับเวลาและหยุดได้อย่างแม่นยำ และยังสามารถหยุดเวลาผ่านเซนเชอร์ หรือการกดปุ่มได้
