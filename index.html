ปัญหาที่ต้องการแก้ไข

Project ต้องการสร้างตัวจับเวลาสำหรับ เกมโชว์ ความบันเทิง การออกกำลังกาย และ การฝึกฝน ความสำคัญของโปรเจคคือ เพื่อจับเวลาให้ตรงความจริงให้ได้มากที่สุด คนที่ได้ประโยชน์ นักออกกำลังกาย ผู้จัดการแข่งขัน นักกีฬา ปัญหานี้แก้ไขได้โดย Project นี้จะใช้ censor ในการตรวจจับและหยุดเวลาทำให้ได้เวลาที่ใกล้เคียงความจริงที่สุด Input เวลาเริ่มต้น ภาพจากกล้อง Output เวลาสิ้นสุด และ ข้อมูลเวลา

ความสำคัญของปัญหา

ปัญหาในการจับเวลาให้แม่นยำมีความสำคัญอย่างยิ่งต่อกิจกรรมหลายประเภทเช่น การแข่งขันต่างๆ การใช้เซ็นเซอร์เป็นแนวทางแก้ไขที่มีประสิทธิภาพสูง เนื่องจากเซ็นเซอร์สามารถให้ข้อมูลที่แม่นยำและรวดเร็ว ทำให้ผลลัพธ์ที่ได้มีความน่าเชื่อถือและสามารถนำไปใช้ประโยชน์ได้อย่างหลากหลาย

ความคืบหน้าโครงการ

รายละเอียดขั้นตอนที่ดำเนินการ

การระบุปัญหา :
เราเริ่มต้นด้วยการค้นคว้าปัญหาทั่วไปที่เกิดขึ้นในการดูแลต้นไม้ โดยเน้นไปที่การรดน้ำที่ไม่สม่ำเสมอซึ่งมาจากการขาดความรู้เกี่ยวกับระดับความชื้นในดิน ซึ่งเป็นปัจจัยสำคัญที่ส่งผลต่อสุขภาพของต้นไม้ โดยเฉพาะสำหรับคนที่มีตารางเวลาที่แน่น


การวางแผนการแก้ปัญหา :
หลังจากเข้าใจปัญหาแล้ว เราวางแผนพัฒนาระบบที่ใช้บอร์ด Arduino และเซนเซอร์วัดความชื้นในดิน เป้าหมายของเราคือสร้างอุปกรณ์ที่สามารถตรวจสอบความชื้นในดินได้อย่างแม่นยำและแจ้งเตือนผู้ใช้เมื่อต้นไม้ต้องการน้ำ


การสร้างต้นแบบ :
ขั้นตอนถัดไปคือการประกอบอุปกรณ์ที่จำเป็น ได้แก่ บอร์ด Arduino เซนเซอร์ และอุปกรณ์อิเล็กทรอนิกส์อื่น ๆ เราทำการเชื่อมต่อและทดสอบวงจรเบื้องต้นเพื่อให้แน่ใจว่าอุปกรณ์ทำงานได้ตามต้องการ


การเขียนโค้ดและการทดสอบ :
เมื่อฮาร์ดแวร์พร้อมแล้ว เราเขียนโค้ดให้ Arduino สามารถอ่านค่าความชื้นและส่งการแจ้งเตือนได้ ขั้นตอนการทดสอบมีการปรับค่าให้เหมาะสมกับสภาพดินและสภาพแวดล้อมที่แตกต่างกัน เพื่อให้ค่าที่ได้มีความเสถียร


การแก้ไขปัญหา :
เราพบปัญหาการตั้งค่าความไวของเซนเซอร์ที่ส่งผลให้ค่าที่อ่านได้ไม่แม่นยำ ในการแก้ไขปัญหานี้ เราได้ปรับตำแหน่งเซนเซอร์และปรับโค้ดเพื่อเพิ่มความแม่นยำ จนกระทั่งได้ค่าความชื้นที่เสถียรและเชื่อถือได้


สถานะปัจจุบัน :
ระบบสามารถตรวจวัดค่าความชื้นในดินและสามารถแสดงสถานะของต้นไม้ผ่านทางหน้าจอ OLED ได้เสร็จสิ้นตามจุดประสงค์ที่เราต้องการแล้ว ในอนาคตเราสามารถพัฒนาเพิ่มการแจ้งเตือนในโทรศัพท์เพื่อความสะดวกสบายของผู้ใช้งานมากขึ้น
